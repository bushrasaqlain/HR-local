
First Get Project by clone:

```bash

git clone https://github.com/LabHazirM/Job-Portal.git

```
This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:
```bash
npm install -global yarn
```
```bash
yarn add next
```
```bash
npm run dev
# or
yarn dev
```
To run the backend open package.json file and add  "server": "nodemon index.js",     in scripts  and run the following command in project folder
```bash
npm install -global nodemon
```
```bash
nodemon index.js
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
